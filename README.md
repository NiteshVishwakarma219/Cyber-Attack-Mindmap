# Phishing-Based Cyber Attack Flow

This project demonstrates how a phishing attack occurs step-by-step.

## Attack Flow:
1. Reconnaissance
2. Phishing Email
3. Credential Theft
4. Initial Access
5. Lateral Movement
6. Data Exfiltration

## SOC Detection Points:
- Email filtering detects phishing
- Login logs detect unusual access
- SIEM tools generate alerts

## Tools Used:
- Draw.io
- Basic SOC Concepts
